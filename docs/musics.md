#Musics
Following endpoints are valid for musics. `X-Access-Key` and `X-Access-Token` are required wherever the endpoint expects 
authorization. 

##Endpoints

###GET `/musics`
* List of musics from Picovico Library

###GET `/me/musics`
* List of user's uploaded musics.

###PUT `/me/musics`
* Upload music.
